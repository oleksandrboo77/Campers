# TravelTrucks — оренда кемперів

Фронтенд частина веб-додатку для компанії TravelTrucks, що займається орендою кемперів. Користувач може переглядати каталог кемперів, фільтрувати їх, додавати в обране, переглядати деталі, відгуки та заповнювати форму бронювання.

## Основний функціонал

- Головна сторінка:
  - Банер з основним закликом до дії
  - Кнопка **View Now** для переходу до каталогу `/catalog`
- Каталог `/catalog`:
  - Отримання даних з публічного MockAPI
  - Фільтрація на бекенді за локацією, типом кузова та обладнанням
  - Пагінація / довантаження через кнопку **Load more**
  - Додавання кемперів в обране, збереження в localStorage
  - Перехід на детальну сторінку по кнопці **Show more**
- Сторінка кемпера `/catalog/:id`:
  - Детальна інформація: назва, рейтинг, локація, ціна, опис, галерея
  - Вкладки **Features** та **Reviews**
  - Features: використання властивостей transmission, engine, AC, bathroom, kitchen, TV, radio, refrigerator, microwave, gas, water (якщо присутні)
  - Vehicle details: form, length, width, height, tank, consumption
  - Форма бронювання з вибором дати (кастомний DatePicker), валідацією та нотифікацією про успішне бронювання

## Використані технології

- Next.js (App Router)
- TypeScript
- Zustand (глобальний стан: список кемперів, фільтри, обране, пагінація)
- Axios (HTTP-запити до MockAPI)
- CSS Modules (стилізація компонентів)

## Робота з API

- `GET /campers` — список кемперів з підтримкою фільтрації та пагінації
- `GET /campers/:id` — деталі окремого кемпера
Фільтрація реалізована на бекенді через query-параметри.
